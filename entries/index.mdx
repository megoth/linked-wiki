# Linked Wiki

This is the frontpage of LinkedWiki. By default, the wiki only contains this page and an [About page](/about). If you
want to make one of your own, simply clone this repo and go wild. (It's all experimental, so I'm not planning to make
it into a NPM package or anything like that - not for now, at least...)

## The idea

To manage a wiki using a set of [MDX](https://mdxjs.com/) components, which when assembled also builds a semantic
knowledge graph based on the content of those components. I'm also aiming to use it for another personal project of
mine that's coming up, so I'm tailoring features around what I need for that project, such as using git to deploy
updates.

I'm building this tool using technologies I'm familiar with (such as TypeScript, Vite, React, and Bulma), as I aim for
it first and foremost to be a personal project. That said, I want to develop it publicly in case people are interested,
and want to help or use it for something of their own.

If you want to know more about the technicalities, check out the [About page](/about).
